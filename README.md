# QuasiHamiltonianRTN
[![Build Status](https://travis-ci.org/matteoacrossi/QuasiHamiltonianRTN.jl.svg?branch=master)](https://travis-ci.org/matteoacrossi/QuasiHamiltonianRTN.jl)

A [Julia][jl] package for evaluating the dynamics of a quantum system affected by one or more sources of [random telegraph noise][rtn] (RTN). It implements a method introduced in [Joynt, R. *et al.*, Int. J. Mod. Phys. B 25, 2115â€“2134 (2011)][paper], that allows to evaluate numerically, with arbitrary precision, the state of the system at any time instant `t`.

## Installation

QuasiHamiltonianRTN is compatible with Julia 0.7 and newer. To install it, access the package manager REPL with `]` and type

     add https://github.com/matteoacrossi/QuasiHamiltonianRTN.jl
